컴포넌트 트리 — 렌더 포함 관계 및 Props

실선 화살표는 렌더 포함 관계, 레이블은 전달되는 props / callback입니다.
NotesProvider는 Context로 하위 트리 전체에 데이터를 공급합니다.

상태 흐름 — UI 상태 vs 데이터 상태 분리

App은 선택/생성 같은 UI 상태만 소유하고, NotesContext는 notes 배열과 서버 통신 상태만 소유합니다.
두 레이어는 섞이지 않습니다.

데이터 시퀀스 — 노트 생성/수정/삭제 흐름

사용자 액션부터 API 호출, 낙관적 상태 업데이트까지 전체 데이터 흐름을 보여줍니다.
에러 발생 시 Context에서 catch하고 컴포넌트는 console.error()로만 처리합니다.
